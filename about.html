About ST-Activities>

This is a project that used the Strava API to pull data for mapping, charting, and create a table with the strava athletes recent activity data. It is meant as an example of the type of work that can be done in JavaScript, Node.js, charts.js, leaflet maps, and more.

Many resourses were used to create this app that are listed on the acknowledgements page. They were invaluable during times of crisis. I would like to point out one resourse and that was YouTuber Franchyze923 . He provided the insight for getting the Stava API data to render successfully to the client(me).
